# 2. Présentation
Avec le même `public static void main(String[] args) {}` nous allons le répéter plusieurs fois à la suite des autres (saut de ligne) pour pouvoir afficher le texte voulu :
```Java
class Main {
   public static void main(String[] args) {
      System.out.println("Coucou !");
      System.out.println("Je m'appelle Camthalion");
      System.out.println("Ma devise est 'Parler peu mais parler bien'.");
   }
}
```
